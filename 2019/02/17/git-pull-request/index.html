【git】创建pull request

DevLog
2019-02-17
git

前言：在使用 Hexo + github 搭建博客平台之后，更换主题为cactus，一款黑色背景的护眼主题，发现存在不完善的地方，所以决定通过 pull request 对其进行完善。

环境：github

步骤：

进入hexo博客目录

先删除themes目录下的cactus主题：rm -rf themes/cactus

fork源项目到自己的仓库

clone新项目到themes目录下

重命名为cactus：mv themes/hexo-theme-cactus themes/cactus
